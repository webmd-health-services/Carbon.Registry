The "Carbon.Registry" module...

# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'Carbon.Registry'
Import-Module -Name 'Carbon.Registry'
```

To install privately:

```powershell
Save-Module -Name 'Carbon.Registry' -Path '.'
Import-Module -Name '.\Carbon.Registry'
```
